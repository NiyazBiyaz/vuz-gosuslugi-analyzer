# Анализатор списков поступления.

Скрипт-анализатор списков поступления в вузы, публикуемых на Госуслугах.

### Инструкция по использованию

> [!IMPORTANT]
> Перед запуском скачайте Python на компьютер. Необходима версия **3.14**, старые не сработают. Качать тут: https://www.python.org/downloads.

1. Скачиваете файлы списков с госуслуг:
    - Заходите на сервис подбора вуза.
    - Находите интересующий вуз и программу обучения.
    - Выбираете конкурсную группу.
    - Открываете таблицу со списками, нажимаете *"Скачать в виде таблицы"*
2. Создаете новую директорию рядом с файлами проекта. Туда кладете скачанные ранее файлы таблицы.

> [!IMPORTANT]
> Для каждого вуза создавайте отдельную папку, иначе списки перемешаются.

3. Запускаете скрипт через команду `python.exe analyzer.py`. Скрипт предложит сначала зарегистрировать вузы на основе созданных в пункте **2.** директорий, потом выбрать количество доступных мест для поступления для каждой выбранной конкурсной группы.

4. Если Вы не подали согласие в данный конкретный вуз, но хотите узнать проходите ли, то добавьте свой код участника после команды запуска. Примерно так: `python.exe analyzer.py 1234567`. В этом случае участник с кодом `1234567` останется в списках даже если согласие не подавал.

> [!NOTE]
> Чтобы результаты списков были точнее, загружайте больше таблиц одного вуза, так будет известно больше выбранных приоритетов на каждого пользователя.
> **Не забывайте обновлять таблицы** почаще. Люди меняют свои заявления и согласия каждый день, а скачанные таблицы - статичны.

### Зачем это нужно?

В этом году (2026) я поступаю в вуз, и у нас есть сервис поступления в Госуслугах. Он достаточно удобен за тем исключением, что таблицы мало что говорят о том, проходишь ты или нет по нужному направлению или в целом поступишь ли если подашь согласие.
У каждого свой приоритет поступления, поэтому но таблица на сайте это не учитывает и отображает голый список подавших заявление.

В большинстве случаев пройти на специальность возможно, просто человек стоящий выше с более высоким баллом уйдет туда, куда он поставил приоритет.
Поэтому чтобы точно понимать в какой вуз **безопасно** оставлять согласие и существует этот проект.

На базе предоставленных пользователем списках создается новый список, где все участники отсортированы в по количеству баллов, а рядом показана конкурсная группа, куда проходит участник.

> [!WARNING]
> Созданные списки не являются гарантией того, проходит ли кто-то в вуз или нет. Верьте результатам из списков, созданных скриптом, на свой страх и риск. См. файл LICENSE для более точной информации.
